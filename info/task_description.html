Enunciado

Ana, uma professora de português, está corrigindo redações dos alunos e quer saber quantas vezes uma letra específica aparece em cada redação para analisar a frequência de uso de certas letras.

Crie uma função que receba uma string e uma letra como parâmetros e retorne quantas vezes a letra aparece na string.

Entrada: Uma string e uma letra.

Saída: Um número inteiro indicando quantas vezes a letra aparece na string.
